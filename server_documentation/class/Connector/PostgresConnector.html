Class: Connector.PostgresConnector

Defined in: modules\postgresConnector\PostgresConnector.coffee

Overview

Hold all the methods used to comunicate with database.

These methods are, except a few of them, the same as API methods

Class Method Summary

  • . (void) getApplications(callback, forceUpdate) Get applications from database
  • . (void) getCorridors(callback, forceUpdate) Get an array of corridors name from database
  • . (void) getEvents(callback, forceUpdate) Get an Array of Events from db
  • . (void) setEvent(callback, event) UPDATE an event in database, tracking by event.id

    Only event.seen and event.deleted are UPDATEed
  • . (void) getOverviewData(callback, forceUpdate) Get an OverviewData Object from database
  • . (void) getHistory(callback, forceUpdate, options) Get an array of Values
  • . (void) getTrend(callback, forceUpdate, options) Get an array of Values
  • . (void) getCalls(callback, forceUpdate) Get a callTree
