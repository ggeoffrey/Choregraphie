Module CallTree

Everything related to the #/calltree view.

The calls tree is a JSON object like this one:

 {
  "nodes": {
   "node1": {
    "name": "node1",
    "value": 10
   },
   "node2": {
    "name": "node2",
    "value": 4
   }
  },
  "links": [
   {
    "source": "node1",
    "target": "node2"
   }
  ]
 }

The getCalls() method of your server module should return an object like this one. The other Node and/or Link objects described in this module are used internally. You should care about theme only for dev purposes.
